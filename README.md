# Explainable AI for Financial Sentiment Analysis
### A Fine-Tuned FinBERT Platform with Integrated Gradients and SHAP Evaluation

**BSc (Hons) Computer Science (AI) — Final Year Project**

## Overview

This project implements an Explainable AI platform for financial sentiment analysis. A fine-tuned FinBERT model classifies financial news headlines as positive, neutral, or negative, whilst Integrated Gradients (IG) and SHAP provide token-level explanations of each prediction. The platform is evaluated through a novel four-metric framework (fidelity, stability, cross-method agreement, robustness) synthesised into a composite Explainability Reliability Score (ERS).

The system is deployed as an interactive Streamlit dashboard supporting real-time single-sentence analysis, side-by-side IG and SHAP visualisation, and batch CSV processing.

## Results Summary

| Model | Accuracy | Macro F1 |
|---|---|---|
| TF-IDF + XGBoost | 91.5% | 0.878 |
| TF-IDF + MLP | 90.3% | 0.864 |
| **FinBERT (fine-tuned)** | **98.5%** | **0.975** |

**Key XAI finding:** Neutral-class predictions receive near-zero fidelity (mean = −0.001) compared to positive (0.880) and negative (0.929), revealing a fundamental limitation of token attribution methods on absence-of-evidence predictions.

## Setup and Installation

**Requirements:** Python 3.11+
```bash
# Clone the repository
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name

# Create and activate virtual environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

---

## Running the Streamlit App
```bash
streamlit run src/app/streamlit_app.py
```

The app will open automatically at `localhost:8501`. The fine-tuned model weights are included in `models/finbert/finbert_phrasebank_allagree/best_model/` so no training is required.

---

## Running the Evaluation Pipeline

If you want to reproduce the evaluation results, run the scripts in this order:
```bash
# 1. Generate data splits (only needed if data/processed/ is empty)
python src/data/load_phrasebank.py

# 2. Run evaluation scripts
python src/evaluation/fidelity.py
python src/evaluation/stability.py
python src/evaluation/robustness_xai_interaction.py

# 3. Compute composite ERS
python src/evaluation/composite_score.py

# 4. Run significance tests
python src/evaluation/significance_tests.py

# 5. Generate plots
python src/evaluation/make_plots.py
python src/evaluation/more_plots.py
```

---

## Re-Training the Model

The fine-tuned model is included and no retraining is needed. If you want to retrain from scratch:
```bash
python src/modeling/finetune_finbert.py
```

Training runs for approximately 3 epochs with early stopping on validation macro F1. Requires ~16GB RAM. No GPU needed — CPU training works but takes longer.

## Dataset

This project uses the **Financial PhraseBank** (Malo et al., 2014), specifically the `sentences_allagree` subset — 2,264 sentences where all 16 annotators agreed on the sentiment label. Accessed via Hugging Face: `takala/financial_phrasebank`.
